Smoke removal in laparoscopic surgery

Demo Application

Trivial noise based smoke generation


explanations to the parameters

  • X - offset: offsets the visible part of the noise on the X-Axis
  • Y - offset: offsets the visible part of the noise on the Y-Axis
  • Size: sets the size of the visible circle of smoke
  • Animated: if checked the smoke will move in a primitive way
  • Color: sets the color of the smoke
  • Alpha: defines how saturated the smoke will overlay the image
  • Blurriness: blurs the edges of the visible circle of smoke to create a more realistic result
